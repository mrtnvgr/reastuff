A bunch of JSFX plugins, enjoy!

## Plugins

### [**Bitcrusher**](./jsfx/bitcrusher.jsfx)

Downsampling, bit reduction plugin with a clipping stage.

### [**Silencer**](./jsfx/silencer.jsfx)

Simple volume trimmer to cut reverb tails.

### [**Tempo sync for all**](./jsfx/tempo_sync.jsfx)

Converts note division to ms and float range.

### [**Transpose by MIDI channel number**](./jsfx/transpose_midi_chan.jsfx)

> new_note_pitch = note_pitch + note_midi_channel;

Made for quick transposing on Arturia Microlab.

### [**MIDI CC -> Parameter**](./jsfx/midi_cc_to_parameter.jsfx)

Exposes selected MIDI CC as sliders for parameter mapping.

### [**Audio -> Parameter**](./jsfx/audio_to_parameter.jsfx)

Exposes 1 audio channel in a parameter, useful for CV.

### [**MIDI CC -> Audio CV**](./jsfx/midi_cc_to_cv.jsfx)

Converts MIDI CC events to audio CV *with batteries included*.

### [**Parameter scale / invert / attenuate / attenuvert**](./jsfx/parameter_utils.jsfx)

Dead simple jsfx for parameter scaling, offseting
